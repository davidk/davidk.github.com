About

31 May 2014

Hello, i’m David. I also go by dave/davidk.

I work on cloud tooling and cloud infrastructure (starting from bare metal on up), mostly tailored to all things Linux.

This page is a collection of projects I do in my spare time: mostly consisting of electronics, software and 3D printables.

Software

GitHub has a bunch of pieces of code i’ve written over the years. It’s all in various formats/versions of Go, Ruby, C, Bash, Python and/or Perl.

I have an interest in small embedded systems, so sometimes embedded distributions that i’ve built end up there.

Electronics

For circuits and PCBs, OSHPark shared projects has a bunch of things i’ve gotten fabricated and released publicly.

The board that I like the most so far:

  • Electroscope - A toy electroscope for visualizing charge, derived from a electrets video by vk2zay.

    Another explaination is available via @rrmutt, who had a set of boards made (and tried it out):

    • Electroscoping The Cat by @rrmutt

The stratum 1 NTP server has gotten attention over the years (and a lot of really great feedback, too):

  • Raspberry Flavored Time (A NTP server on your Pi, tethered to a GPS unit)

3D Printables

I build a lot of enclosures for breakout boards, as my workspace gets dusty. Maybe you’ve seen or used one in the wild? (If you have, let me know?)

Check out Thingiverse and YouMagine have release-level designs.

Most things will appear on Collector3 first since they don’t need full documentation + photos and will act as a place where designs can be viewed/verified as they evolve over time.
